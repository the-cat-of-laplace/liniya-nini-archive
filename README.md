# 栗妮亚_nini档案馆

一个无需构建工具的静态人物百科站，资料以栗妮亚_nini的哔哩哔哩公开主页与投稿为主要来源。

在线浏览：https://the-cat-of-laplace.github.io/liniya-nini-archive/

## 打开方式

直接使用浏览器打开 `index.html`。页面不依赖网络即可浏览；跳转到B站原始来源时需要联网。

## 功能

- 条目内即时搜索
- 自动高亮当前目录章节
- 浅色与深色主题
- 五组衣装档案与红黑衣装 / 荆棘女巫3D索引专项记录
- 原始投稿日期、BV号、制作署名与名称证据边界
- 13张新增衣装图片，包含用户提供的原图原样副本
- 本地资料草稿、JSON导出与清理
- 桌面、平板和手机响应式布局

## 文件

- `index.html`：百科内容与页面结构
- `styles.css`：视觉与响应式样式
- `app.js`：搜索、主题、目录与本地草稿功能
- `assets/`：来自栗妮亚_nini公开B站页面的头像与投稿封面
- `assets/outfits/`：初始衣装、红黑衣装、荆棘女巫3D、樱花小兔、弥赛亚与芭蕾兔的图像资料
- `assets/archive/site-hero-preserved.png`：用户提供的首屏截图原样存档，未重编码

## 衣装命名原则

- “弥赛亚”是原始投稿明确使用的角色名。
- “荆棘女巫”来自相关剧情片段标题；红黑模型展示与两支3D投稿没有把它声明为统一的正式衣装或模型名。
- “樱花小兔”“芭蕾兔”是用户确认并用于本站检索的称呼；在找到本人公开命名前，页面持续标注“公开名称待考”。
- 2025年B站个性装扮“栗妮亚·春日来信”不作为樱花小兔衣装来源。

公开数据快照日期为 2026-07-22。粉丝、播放等数字会随平台更新而变化。
