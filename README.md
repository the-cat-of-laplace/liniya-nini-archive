# 栗妮亚_nini百科

一个无需构建工具的静态人物百科，整理栗妮亚_nini在哔哩哔哩公开发布的账号资料、创作作品、直播活动、角色世界观、形象衣装与制作署名。

在线页面：https://the-cat-of-laplace.github.io/liniya-nini-archive/

## 使用

直接用浏览器打开 index.html 即可阅读。页面本身不依赖网络；跳转B站原始投稿和主页时需要联网。

## 内容范围

- 账号基本信息、公开数据快照与主要内容方向
- 主账号“直播回放”系列、“栗妮亚的录播号”（UID 3546381977389256）及其“2023年度直播回放”合集的公开信息
- 翻唱、自我介绍曲、OC角色PV与弥赛亚故事等代表作品
- 初期形象、红黑衣装及其3D版本、樱花小兔（暂称），以及白兔在《融化的第五个夏天》中的形象
- 栗妮亚、白兔、樱花小兔（暂称）、弥赛亚等角色与世界观线索，以及按具体作品整理的制作与合作署名
- 公开封面、模型展示画面和动态画面图集

页面中的粉丝、播放、投稿和回放数量均为记录日期的公开快照，会随平台变化；回放数量按账号分别统计，不将主账号系列与录播号合集相加。角色名已经明确时优先使用角色名，未见公开正式名称的角色形象或衣装会保留原始投稿标题并标注为非官方暂称；已确认的角色身份与尚未公开的衣装名称分开记录，角色关系不超出来源明确表达的范围。

## 文件

- index.html：百科正文、目录、来源链接与图集
- styles.css：视觉样式、暗色主题与响应式布局
- app.js：条目搜索、主题切换、目录高亮、阅读进度与链接复制
- assets/：头像、投稿封面、模型展示图和衣装相关图片
- assets/archive/site-hero-preserved.png：项目早期页面截图原文件，作为历史素材保留
