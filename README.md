# 栗妮亚_nini档案馆

一个无需构建工具的静态人物百科站，资料以栗妮亚_nini的哔哩哔哩公开主页与投稿为主要来源。

在线浏览：https://the-cat-of-laplace.github.io/liniya-nini-archive/

## 打开方式

直接使用浏览器打开 `index.html`。页面不依赖网络即可浏览；跳转到B站原始来源时需要联网。

## 功能

- 条目内即时搜索
- 自动高亮当前目录章节
- 浅色与深色主题
- 本地资料草稿、JSON导出与清理
- 桌面、平板和手机响应式布局

## 文件

- `index.html`：百科内容与页面结构
- `styles.css`：视觉与响应式样式
- `app.js`：搜索、主题、目录与本地草稿功能
- `assets/`：来自栗妮亚_nini公开B站页面的头像与投稿封面

公开数据快照日期为 2026-07-22。粉丝、播放等数字会随平台更新而变化。
